安卓源码的下载 编译

真心觉得下载个源码都能这样，还能说啥，f*ck the GFW!!! 确实比较麻烦，简单记录一下。

环境： win7 + vmware(ubuntu 10.04)

0x00:

先安装一些必要的组件。
可以先更新一下源，改成163的。

参考：http://blog.csdn.net/xinyuwuxian/article/details/8971210

sudo apt-get install git-core gnupg
sudo add-apt-repository ppa:ferramroberto/java
sudo apt-get update
sudo apt-get install sun-java6-jre sun-java6-plugin
sudo apt-get install sun-java6-jdk
sudo apt-get install flex bison gperf libsdl-dev libesd0-dev libwxgtk2.6-dev build-essential zip curl

0x01:

mkdir ~/bin
export PATH=$PATH:~/bin
curl http://git-repo.googlecode.com/files/repo-1.12 > ~/bin/repo
chmod a+x ~/bin/repo

0x02:

修改repo的第五行，https改为http

0x03:

mkdir ~/android-src
cd ~/android-src/
repo init -u http://android.googlesource.com/platform/manifest -b android-2.2.2_r1
输入自己的name，mail

repo sync

0x04:

中间可能会卡住不动，ctrl+z，然后重新repo sync即可。
